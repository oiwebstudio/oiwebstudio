# OI Studio — Diseño y Desarrollo Web

Sitio web profesional de OI Studio, especializado en webs modernas, rápidas y efectivas para negocios locales del País Vasco.

## 🌐 Características

- **Diseño premium y responsivo** — Funciona perfectamente en móvil, tablet y desktop
- **Rendimiento optimizado** — Carga rápida, imágenes optimizadas, código limpio
- **Multiidioma** — Español (ES) y Euskera (EU) con toggle integrado
- **SEO mejorado** — Datos estructurados, meta tags, sitemap.xml y robots.txt
- **Accesibilidad** — Cumple estándares WCAG, navegación accesible
- **Animaciones fluidas** — Efectos de hover, scroll y transiciones CSS
- **Formulario de contacto** — Integrado con WhatsApp

## 🚀 Visualización Local

Para ver la web en tu máquina:

```bash
# Navega a la carpeta
cd "mi web"

# Sirve con un servidor local (Node.js)
npx http-server

# O usa Python
python -m http.server 8000
```

Abre `http://localhost:8000` en tu navegador.

## 🛠️ Antes de Publicar

1. **Rellenar datos legales** en `legal.html` (campos marcados en amarillo)
2. **Actualizar dominio** en `sitemap.xml` y `robots.txt` (reemplazar placeholder)
3. **Configurar URLs absolutas** en Open Graph meta tags (og:image, og:url)
4. **Verificar email de contacto** — Actualmente: contactoiwebstudio@gmail.com
